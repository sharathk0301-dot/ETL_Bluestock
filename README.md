# Nifty100 Analytics Dashboard

A full-stack market research and analytics platform for exploring **Nifty 100 companies** through interactive dashboards, company-level fundamentals, comparative financial analysis, and modern web visualizations.

This project was built to make equity research more structured, visual, and accessible. Instead of reading raw statements in isolation, the platform helps users move from a market-wide view to sector-level exploration and finally to individual company insights. 

## Overview

Nifty100 Analytics Dashboard is a web-based application designed to analyze listed companies in the Nifty 100 universe using a combination of: 

- financial fundamentals,
- growth metrics, 
- profitability indicators, 
- leverage ratios, 
- cash flow trends,
- interactive visual storytelling. 

The application combines a modern frontend experience with structured financial data presentation so that users can quickly understand which companies are leading, which are improving, and which require deeper investigation.

## Features

### 1. Market Overview
- View the broader Nifty 100 landscape in one place. 
- Identify leading companies based on revenue, net profit, and other core metrics.
- Understand high-level trends before drilling into specific stocks. 

### 2. Company Fundamentals
- Explore company-wise financial indicators such as: 
  - Revenue 
  - Net Profit 
  - ROE 
  - Debt-to-Equity 
  - Revenue CAGR 
  - Other balance-sheet and profitability metrics 
- Compare companies across multiple dimensions using structured visuals. 

### 3. Cash Flow Analysis
- Visualize **CFO, CFI, CFF, and Net Cash Flow** over time.
- Use creative multi-series charts to understand how operating, investing, and financing activities interact.
- Spot consistency, volatility, and long-term capital allocation patterns.

### 4. Company Drill-Down Pages
- Move from index-level analysis to company-level detail. 
- Review financial performance, trends, and supporting visuals in a single flow. 
- Build a more complete picture of business quality and financial strength.

### 5. Interactive Web Experience
- Responsive design for better usability across devices. 
- Reusable UI components and clean routing.
- Modern dashboard styling with a focus on readability, hierarchy, and user experience. 

### 6. Data-Driven Research Workflow
- Designed as more than a static dashboard.
- Structured to support a research-oriented workflow where users can:
  - scan the market,
  - compare businesses, 
  - review financial trends, 
  - and extract decision-ready insights.

## Architecture Approach

The application was designed around a layered experience: 

1. **Overview layer** – understand the index at a high level.
2. **Comparison layer** – compare companies using charts and metrics. 
3. **Insight layer** – analyze deeper financial trends such as cash flow and leverage,
4. **Detail layer** – inspect individual company-level performance.

This structure makes the dashboard useful not only for visualization, but also for guided analysis. 

## Design Philosophy

A major focus of the project was usability. 

The dashboard was designed with the following principles:

- clean and readable visual hierarchy, 
- minimal clutter, 
- metrics that matter most placed first,
- charts designed for interpretation rather than decoration, 
- and an interface that supports fast exploration. 

Special attention was given to creating more creative visual components for financial data, including stylized trend views and modern dashboard panels inspired by research and analytics tools.
